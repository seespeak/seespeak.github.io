What is it?

SeeSpeak is a 7" tablet customised with our bespoke software. It is a single purpose device with no other distractions.

What does it do?

SeeSpeak displays photographs of contacts. With just one tap one the photograph, the users launches a Skype video call.

Why is it needed?

SeeSpeak has been developed to enable communication between generations, especially those who find technology challenging.

Communication - Connection - Conversation

Faced with this ageing population, and a more mobile generation, many of us find ourselves living miles from our parents and extended family. Technology is fast-moving and increasingly complicated, further alienating elderly relatives.

SeeSpeak has been designed to be easy to set up and easy to use, removing any worry for the technology wary older user.

  • Make a video call with one tap
  • Uses Skype so you can easily connect to an existing large network of friends and family
  • Contacts are displayed as easy to see large photos and names
  • Photos are automatically downloaded from Skype or you can set your own
  • Comes with table top stand an long power lead so it can live in a prominent place in the home and be always charged
  • Designed to be always on and ready to make and receive calls
  • Screen dims to low level when not being used so not to be obtrusive
  • The SeeSpeak software turns the tablet into a single purpose device, preventing the user from accidentally starting other applications
  • The user is shielded from the confusing pop up message and notifications that normally occur on other tablets.
  • Quick to set up by a family member and easy to instruct the end user

How does it work?

Before Starting

A friend or family member of the user sets up a Skype account, adds in contacts and arranges access to Wi-Fi

Turning on

On turning on SeeSpeak for the first time, the helper is guided through connecting to the Wi-Fi and signing in to Skype

Ready to use

With simple guidance, the end user can now make video calls with maximum ease; stress free technology!

Easily Accessible

Living in a prominent place and always on, calls can be made and received any time

What's in the box?

  • Acer 7 inch Android Tablet.
    The actual model may vary, but the minimum specifications will be a 600 x 1024 pixel screen, Dual Core 1.2 GHz, 1 GB Ram, 0.3M pixel front facing camera
  • Table-top stand
  • 1.8m USB cable and charger
  • Full paper instruction manual
